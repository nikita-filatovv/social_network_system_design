## Функциональные требования
1. Пост с небольшим описанием, фото и привязкой к локации
    - Создание
    - Удаление
    - Привязка к локации
2. Реакции на публикации
    - Оценка поста (like, dislike)
    - Изменение реакции
    - Получение списка поставивших реакцию
3. Комментарии
    - Добавление
    - Удаление
    - Загрузка комментариев под постом (пагинация 10)
4. Система подписок
    - Действие подписки
    - Действие отписки
    - Просмотр списка подписок
    - Просмотр списка подписчиков
5. Поиск популярных мест для путешествий (Локаций)
    - Определение популярности мест (к которым большего всего постов)
    - Поиск популярных мест (пользователем)
6. Лента (пагинация 10)
    - Просмотр своей ленты
    - Просмотр ленты отдельного пользователя
    - Просмотр общей ленты (на основе подписок в обратном хронологическом порядке)
    - Просмотр ленты к Локации
7. Регистрация
    - Имя пользователя
    - email
    - пароль
    - avatar
8. Вход
    - email+password

## Нефункциональные требования
1. DAU - 10000000
    - Линейный рост до этого показателя на дистанции года
    - Через год планируется дальнейший рост (масштабируема)
    - Примерные показатели активности одного пользователя:
        1. 24 поста от пользователя в год - 0,07 (грубо в день)
        2. 20 постов в день читает пользователь
        3. 20 реакций в день 
        4. 3 комментария в день
        5. 2 поиска в сутки
        6. 10 загрузок ленты (любые, основной, пользователя и тд)
2. География - страны СНГ
    - Без плана выхода на глобальный рынок
3. Платформы
    - Браузер (используется на мобилке и компе)
4. Лимиты
    Посты:
    - 5 фото в посте
    - 1000 символов - длинна поста
    - 20 мб - размер файла (фото)
5. SLA (latency)
    - 4 секунды - создание поста
    - 5 секунд - загрузка фото
    - p95 < 100API latency
    - >1 секунды - добавление реакции
    - 2 секунды - добавление комментария
6. Доступность системы: ```99,95%```

## Оценка нагрузки
4. RPS (примерно)
    - Посты (write) - 2,5
    - Посты (read) - 1150
    - Реакции (post) - 1150
    - Реакции (read) - 1150
    - Комментарии (post) - 1150
    - Комментарии (read) -  23000  
5. Traffic (примерно)
    - Посты (write) -  50кб * 2,5 = 125кб/с
    - Посты (read) - 65кб * 1150 = 73мб/с
    - Реакции (post) - 0.2кб * 1150 = 230кб/с
    - Реакции (read) - 0.2кб * 1150 = 230кб/с
    - Комментарии (post) - 3кб * 1150 = 3.4мб/с
    - Комментарии (read) - 3кб * 23000 = 67мб/с
6. Доступность системы - 99,9%
